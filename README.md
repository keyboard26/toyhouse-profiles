# Toyhouse HTML Layouts

A collection of **custom character profile layouts** designed for **Toyhouse**.  

These layouts are built with **HTML and inline CSS only**, following Toyhouse’s code editor limitations. They are intended **only for use within Toyhouse** and may not display correctly outside the site.

## Features
- Pure HTML + inline CSS
- Safe for Toyhouse’s editor
- Ready to paste directly into profiles
- Each layout has its own folder and description
