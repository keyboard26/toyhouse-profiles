# Butterflies

This layout is built with **HTML and inline CSS only**, designed specifically for **Toyhouse**.  
It is intended to be pasted directly into a Toyhouse character profile.  

> Note: It will **not render correctly outside Toyhouse** due to site-specific CSS classes and limitations.


View the properly rendered and interactive version [here](https://toyhou.se/35305388.f2u-html-butterflies)

This was created using the [Coders' Quarters Autumn Challenge](https://toyhou.se/~world/82691.coders-quarters/page/163684.autumn-challenge). Coders' Quarters releases seasonal challenges and prompts to encourage coders to experiment with new techniques.
