# Hollow Knight

This layout is built with **HTML and inline CSS only**, designed specifically for **Toyhouse**.  
It is intended to be pasted directly into a Toyhouse character profile.  

> Note: It will **not render correctly outside Toyhouse** due to site-specific CSS classes and limitations.


View the properly rendered and interactive version [here](https://toyhou.se/33465736.f2u-html-hollow-knight)

<br>


This layout was created to mimic the look of the inventory from Hollow Knight (the best game ever made). It is intended for Hollow Knight fan characters, but can be used for anything.
